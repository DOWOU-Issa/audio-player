Un lecteur audio/vidéo professionnel avec extraction audio intégrée, recherche YouTube et visualiseur DJ animé.


##  Fonctionnalités

###  Lecture locale
- Charger un dossier complet de musique
- Sélectionner des fichiers audio/vidéo individuels
- **31 formats supportés** : MP3, WAV, MP4, AVI, MKV, FLAC, M4A, etc.

###  Recherche YouTube
- Recherche directe depuis l'application
- Résultats avec miniatures
- Lecture instantanée dans un popup déplaçable

###  Extraction audio
- Extraire l'audio de n'importe quelle vidéo
- Conversion automatique en MP3
- Téléchargement du fichier extrait

###  Interface DJ
- Égaliseur animé en temps réel
- Dégradé de fond réactif à la musique
- Contrôles complets : Play/Pause, Shuffle, Repeat
- Vitesse de lecture : 0.5x à 2x
- Curseur de volume dynamique

##  Installation

```bash
# Cloner le dépôt
git clone https://github.com/DOWOU-Issa/audio-player.git
cd audio-player

# Installer les dépendances
npm install

# Démarrer le serveur de développement
npm run dev
